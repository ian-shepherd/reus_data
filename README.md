## reus
Repository for the purpose of storing various soccer datasets to support the [`reus`](https://github.com/ian-shepherd/reus) package. Specific shoutout to [worldfootballR](https://github.com/JaseZiv/worldfootballR) and their associated [dataset](https://github.com/JaseZiv/worldfootballR_data). Odds data is courtesy [football-data](https://www.football-data.co.uk/).
